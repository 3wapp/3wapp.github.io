
## 查看函数调用

找到函数定义的位置，按住`ctrl`, `左键`点击函数名，显示所有调用这个函数的代码

## 常用快捷键

| | |
| --- | ---- |
| Ctrl + alt + shift + u | 新建窗口生成一个 UML 图|
| Ctrl + alt + u | 临时窗口生成一个 UML 图|

生成UML图时，要将鼠标放在定义的`类名称`上面，然后在按快捷键
